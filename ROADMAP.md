# Arc Merchant — Public Roadmap

This is a living document. Updated as priorities shift.

---

## ✅ Q1 2026 — Foundation (Complete)

- [x] x402 paywall middleware for paywalled content endpoints
- [x] Circle Developer-Controlled Wallets integration
- [x] Payment facilitator server (on-chain settlement on Arc L1)
- [x] Real-time merchant dashboard (revenue, payments, wallet balance)
- [x] MCP server for Claude Code agents
- [x] Framework-agnostic buyer agent SDK (Vercel AI, Google GenAI adapters)
- [x] Demo buyer agent with built-in chat UI
- [x] Deployed to Railway — live at https://arc-merchant-production.up.railway.app
- [x] 🏆 2nd Place — Agentic Commerce on Arc Hackathon

---

## 🔄 Q2 2026 — Clarity & Usability

- [ ] Clean buyer/seller code split (clear demo narrative)
- [ ] Multi-article content management (add/remove articles without code changes)
- [ ] Webhook support — notify merchant when payment lands
- [ ] Better onboarding — clear setup path for new merchants
- [ ] Custom pricing per article/endpoint
